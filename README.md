# JBNU_BigData_Analyze
<pre>
전북대학교 소프트웨어공학과\
빅데이터 분석 수업\
지도교수 : 류덕산\
1팀 : 김민규(팀장), 김원배, 김범수, 최종민
</pre>

### 과제 내용
<pre>
Socialus.com 의 데이터를 이용하여 영상 장르 별 순위권 채널들의 제목과 조회수, 좋아요/싫어요를 분석하여 \
제목에 들어간 표현들이 조회수에 끼치는 영향력을 점수로 표현하였다. \
이를 통해 어떠한 표현을 사용하면 조회수를 높일 수 있는지 파악한다. 
</pre>
